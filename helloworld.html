Abosulute vs Relative Paths

Absoulute paths always include a specific file and server..."http://blahblah.com"

Anytime you link to another site, it should be an absolute path

Relative paths ask for a file, but not a specific server....href="contacts.html".

Since no server name is specificed, the browser assumes you mean the server that your page is on.
